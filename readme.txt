2.首页路径：http://localhost/ztt/goods/index.html
3.技术栈：（1）原生js
          （2）jq插件
	  （3）使用es6语法
          （4）Bootstrap插件
          （5）使用sass转css    
          （6）gulp工具
	  （7）使用github，做版本管理
4.项目使用方式：
	（1）建立站点，名为ztt，打开首页网址		http://localhost/ztt/goods/index.html
	（2）首页有轮播图，三级菜单，缓慢楼梯效果，商品由数据渲染
	（3）首页未登录无法进入购物车，点击注册，注册成功后进入登录页面，成功登陆后重新进入首页，此时可以进入购物车页面
	（4）点击三级菜单中的女装，进入商品列表，由数据渲染，使用图片懒加载
	（5）点击商品，进入商品详情，由数据渲染，具有放大镜，选项卡功能，点击加入购物车，去购物车可查看购物产品
	（6）购物车中的商品可以进行删除和修改
